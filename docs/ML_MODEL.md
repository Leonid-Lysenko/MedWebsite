# Документация ML-модели

## Общие сведения

**Модель**: Logistic Regression  
**Назначение**: Классификация заболеваний по симптомам  
**Библиотека**: Scikit-learn  
**Формат**: Joblib

## Технические характеристики

- **Классифицируемые заболевания**: 429
- **Обрабатываемые симптомы**: 304  
- **Объем обучающей выборки**: 240,000+ случаев
- **Точность (accuracy)**: 86.85%

## Гиперпараметры

```python
{
    'C': 9,

    'max_iter': 1000

```

## Интеграция с системой

**Загрузка модели:**
```python
import joblib
model = joblib.load('ml/models/lr_model_full_data.joblib')
```

**Процесс предсказания:**
```python
# Вход: бинарный вектор симптомов
# Выход: вероятности топ-5 заболеваний
probabilities = model.predict_proba([symptom_vector])[0]
top5_diseases = model.classes_[np.argsort(probabilities)[-5:][::-1]]
```
